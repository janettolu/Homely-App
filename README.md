# Homely

Homely is a plaform that connect renters with verified agents, landlords and artisans.

## Problem
Renting a home in many cities can be very stressful and unsafe
People rely on roadside agents, random Whatsapp groups, or social media posts, which often lead to fake listings, wasted time, agent fraud, and scams
There is also no single verified source to connect with trustworth agents or artisans for maintenance.
Landlords also lose trust and revenue. 

## Solution
Homely builds trust into the rental system while making the process faster and more organised by bringing everything into one platform:
- Verified property listings from licensed agents and landlords
- Connects renters to verified agents
- Enables secure in-app communication between renters and agents.
- Booking and scheduling of house viewing
- Digital payment and rent receipts for transparency
- Links renters to vetted artisans for post-rental services



## Target User 
- Renters looking for safe, fast rentals
- Local real estate agents seeking online platform 
- Landlords wanting transparent transactions
- Artisans offering post-rental services to tenant

## Key features
- Verified profiles: all agents and landlord are identity verified before listing go live
- Smart search and filter: search by location, address, type, or price range
- In app chat and scheduling viewing with digital confirmation
- Rent payment and e-receipt
- Homely connect: feature linking renters to vetted artisans for repairs after moving in


## Tech (high level)
Mobile first: React Native (frontend), Node.js + Express (API), PostgreSQL (database), payment gateway (Flutterwave), Map API (property location)
